Requirement:
------------------------------------------
	Windows XP/7 or OSX
	Adobe Flash CS 5
	Adobe Flash CS 4 ( known issues see FAQ )
	Adobe Air runtime

------------------------------------------
Upgrading:
------------------------------------------
	1. Install the latest uniSWF Converter
	in the /uniSwf/installer folder.


------------------------------------------
Installation:
------------------------------------------
	1. Extract the uniswf package into the 
	root of your project, a uniSWF folder 
	should be present once complete.
	
	2. Install the adobe air runtime, you 
	can download it from the following 
	address: 
		http://get.adobe.com/air/
		
	3. Install the uniSwf converter 
	application. double click on the 
	uniSwf.air application in the 
	/uniSwf/installer folder.
